# Baskal — Landing Page

Marketing landing page for **Baskal**, a mobile app that tells you exactly what
groceries to buy based on your **budget, calorie goals, nutrition goals, and
preferred supermarket**.

**Set your goals. Get the exact groceries you need.**

🔗 **Live:** https://baskal.vercel.app

## Stack
- Static HTML/CSS, zero dependencies, zero build step.
- Fonts: Manrope (display) + Switzer (body), self-hosted.
- All produce cut-outs, lifestyle photos and the basket hero were generated with
  **Higgsfield** (Nano Banana Pro) and background-removed for the floating elements.

## Structure
```
index.html            # the page
assets/img/*          # produce cut-outs + lifestyle photos
assets/fonts/*        # Manrope + Switzer woff2
```

## Deploy
Static — deploy the folder as-is (e.g. `vercel --prod`).
